# 🌍 Jornada Viagens

Uma landing page responsiva com temática de viagens, desenvolvida utilizando apenas **HTML5** e **CSS3**. O projeto foi criado com foco em uma interface moderna, organização do conteúdo e adaptação para diferentes dispositivos.

## 🚀 Demonstração

🔗 **Acesse o projeto:**

## 💻 Executando o projeto

Clone o repositório:

```bash
git clone https://github.com/DiegoOGoncalves/jornada-viagens.git
```

Entre na pasta:

```bash
cd jornada-viagens
```

Depois, abra o arquivo `index.html` no navegador.
